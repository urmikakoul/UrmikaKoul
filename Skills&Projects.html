Product & Project Mgmt.

Product Canvas - Aggieland Humane Society

In this project, I created a Product Canvas for Aggieland Humane Society. The system will help Aggieland Humane Society enhance their current system to manage adoptions, various types of volunteers, employees, donation and scheduling of events. This new system will help them better manage and leverage their data as well as improve the user experience of volunteers, donors and adopters by utilizing cutting edge technology like Artificial Intelligence.

Product Canvas - Brazos Valley Farmer's Market

In this project, my team and I created this Product Canvas for Brazos Valley Farmer's Market. This system for the Brazos Valley Farmers Market will help manage vendors, as well as local farmers and producers to grow their business. The system will help the market build a community. The goal is to help the organization manage the Brazos Valley Farmers Market by streamlining their processes and building a system that can reduce manual efforts. The system will provide a platform for vendor management and shift most of the paperwork online. It will help the treasurer manage the payments better by introducing direct online payments. It will also help the board of directors make better decisions based on vendor data and reports from various activities of the market.

Artificial Intelligence

Skin Cancer Classification

Link to Github

In this project, I implemented CNN, transfer learning, cyclic learning rate, and learning rate finder using Python, Keras, TensorFlow to classify images in 7 types of skin lesions with 83% accuracy. The dataset has 17705 images. The pre-processing of the images involves resizing the images to (150,150) and normalizing the images. Data Augmentation is applied on the images, to increase the diversity of data available for training models, without collecting more data. Transfer Learning is employed in the model. The base model which is the DenseNet 201 model is taken and a neural network with a few layers is added on top of this base model. All the layers of the base model, that is the DenseNet 201 model are trained along with the layers of the neural network which is added to this base model. The optimizer that is used is Adam. Adam realizes the benefits of both AdaGrad and RMSProp. The loss function used is categorical cross entropy. The commonly used loss function for multi-class classification. Learning rate finder is employed. It is an algorithm that can be used to automatically find optimal learning rates for the deep neural network. From this we obtain the maximum and minimum learning rate for employing in the Cyclic Learning Rate implementation. Using Cyclical Learning Rates leads to faster convergence and with fewer experiments/hyperparameter updates. By using cyclic learning rate and collecting more images for our dataset the weighted precision, weighted recall, weighted F1-Score, weighted ROC-AUC score and balanced multi-class accuracy are all achieving good values. Weighted metrics are used for imbalanced datasets, and hence these metrics are being used to evaluate the model.
